# Design Mockups

This directory contains interactive HTML/CSS mockups showcasing the improved design and UX for the All Clued In crossword website.

## Files

- **homepage-mockup.html** - Homepage design showing mobile and desktop views
- **crossword-page-mockup.html** - Crossword solving page design with all features

## How to Use

1. Open any of the HTML files in a modern web browser
2. Use the toggle buttons at the top to switch between mobile and desktop views
3. The mockups are interactive and demonstrate:
   - Visual design improvements
   - Responsive layouts
   - Component styling
   - Color scheme and typography
   - User interface patterns

## Features Demonstrated

### Homepage Mockup
- Clean, minimalist design
- Series grouping with cards
- Chronological crossword listing
- Responsive grid layout
- Hover states and interactions
- Loading skeleton states (commented out, can be enabled)

### Crossword Page Mockup
- Interactive crossword grid
- Current clue display
- Clue sidebar with highlighting
- Action buttons (Check, Reveal, Clear)
- Mobile-optimized layout
- Desktop side-by-side layout
- Visual feedback for selected/highlighted cells

## Design Principles Applied

All mockups follow the comprehensive design style guide and incorporate:

- **Color Palette**: Primary blue, accent colors (teal, yellow, purple)
- **Typography**: System font stack with proper sizing scale
- **Spacing**: Consistent spacing system (xs, sm, md, lg, xl, 2xl)
- **Shadows**: Subtle elevation for depth
- **Border Radius**: Rounded corners for modern feel
- **Responsive Design**: Mobile-first approach with breakpoints
- **Accessibility**: High contrast, clear interactive states

## Browser Compatibility

These mockups work best in modern browsers:
- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
